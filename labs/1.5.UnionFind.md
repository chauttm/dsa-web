# 1. Chạy chương trình mẫu sử dụng thư viện UF (Union-Find) sau đây.
```Java
public class UFClient { 
   public static void main(String[] args) { 
      int N = StdIn.readInt(); 
      UF uf = new UF(N); 
      while (!StdIn.isEmpty()) { 
         int p = StdIn.readInt(); 
         int q = StdIn.readInt(); 
         if (!uf.connected(p, q)) { 
            uf.union(p, q); StdOut.println(p + " " + q); 
         } 
      } 
   }
}
```
# 2. UFClient2
Sử dụng thư viện UF tại bộ algs4, hãy viết chương trình đọc và thực hiện các lệnh union cho đến hết input hoặc đến khi toàn bộ hệ thống đã liên thông.
Input: đầu tiên là số đối tượng N, tiếp theo là các cặp (số lượng không xác định) các đối tượng p và q (số nguyên từ 0 đến N-1).
Với mỗi cặp đối tượng, nếu chúng chưa liên thông, chương trình cần chạy lệnh union giữa chúng.
Nếu sau lệnh union này mà toàn bộ N đối tượng đã nằm trong cùng một thành phần liên thông thì chương trình in ra số thứ tự của cặp đối tượng đó trong chuỗi các cặp input. Sau đó kết thúc.
Nếu chương trình đã đọc và thực hiện tất cả các cặp số mà vẫn chưa liên thông hoàn toàn thì chương trình in ra “FAILED”.
